# Runners QC App

A web application for viewing and managing runners (Runners QC).

## Key Features

- **Next.js 15** with App Router
- **AWS Cognito Authentication** with advanced cookie management
- **GraphQL API** with TypeScript codegen
- **PWA Support** with offline capabilities
- **Redux + Redux-Saga** state management
- **Veterinary/Equine Medical** tracking focus
